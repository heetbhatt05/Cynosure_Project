# Cynosure - Backend Server

This directory contains the backend server for the Cynosure AI Career Advisor application. It's built with Node.js, Express, and MongoDB.

## Prerequisites

- Node.js (v18 or higher recommended)
- MongoDB (a local instance or a cloud-based service like MongoDB Atlas)
- `npm` or `yarn` package manager
